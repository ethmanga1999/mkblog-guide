# mkblog-guide

一个用于归档和发布独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放、整理和展示多个独立编写的 HTML 页面。这些页面可以包含静态内容、交互演示、技术笔记或其它形式的网页资料。仓库本身不绑定特定主题或用途，仅作为 HTML 文件的托管与索引空间。

## 目录说明

```
mkblog-guide/
├── pages/          # 存放所有独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/         # 公共静态资源（CSS、JS、图片等）
├── index.html      # 仓库首页或导航页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，可包含自包含的样式与脚本。
- `assets/`：若页面间共享资源，可统一放置于此目录。
- `index.html`：可选，用于提供页面列表或快速导航。

## 页面归档说明

- 每个 HTML 页面应保持独立完整，可脱离仓库环境直接打开浏览。
- 页面文件名建议使用有意义的英文或拼音，避免特殊字符。
- 若页面依赖外部资源（如字体、库），请尽可能使用相对路径或自托管版本。
- 如需对页面进行分组或标注，可在 `pages/` 下建立子目录。

## 维护说明

- 欢迎通过 Pull Request 添加新的 HTML 页面，或对现有页面进行修正。
- 请确保新页面不包含恶意代码、跟踪脚本或违反开源社区规范的内容。
- 仓库维护者会不定期检查页面链接与资源有效性。
- 如需删除或归档旧页面，请提交 Issue 说明原因。
